## 爬楼梯

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？


作者：力扣 (LeetCode)
链接：https://leetcode.cn/leetbook/read/top-interview-questions-easy/xn854d/

```python
def climbStairs(self, n: int) -> int:
    if n < 3: return n
    i, j = 1, 2
    while n > 2:
        i, j = j, i + j
        n -= 1
    return j
```
